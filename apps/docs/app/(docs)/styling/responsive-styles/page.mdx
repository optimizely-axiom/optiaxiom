# Responsive Styles

`Box` component support responsive styles for all layout related props.

<Scale
  valueLabel="Minimum width"
  themeKey="screens"
  values={{ base: "0px", ...tokens.screens }}
/>

## Documentation

### Object notation

Use the object prop notation where the key is the breakpoint name.

```tsx
import { Group } from "@optiaxiom/react";

export function App() {
  return <Group flexDirection={{ base: "column", sm: "row" }}>...</Group>;
}
```

### Array notation

Use the array prop notation with the elements in order of condition `[base, sm, md]`.

```tsx
import { Group } from "@optiaxiom/react";

export function App() {
  return <Group flexDirection={["column", "row"]}>...</Group>;
}
```
