# Icons (Optimizely staff only)

We use a licensed **Font Awesome** icon kit with our Optimizely Design System.

<Alert intent="warning" mt="16">
<p style={{ margin: 0 }}>**Optimizely staff** should always use icons from our **Font Awesome** icon kit.</p>

<p style={{ margin: 0 }}>**Third party developers** can use any available icon library to build extensions.</p>
</Alert>

## Installation

### Install package

Make sure you have access to our private package repository and install the `@optimizely/axiom-icons` package.

```sh npm2yarn
npm install @optimizely/axiom-icons
```

### Basic usage

Simply import the components and use them in your application.

```tsx
import { IconMessages } from "@optimizely/icons";

<IconMessages />;
```
