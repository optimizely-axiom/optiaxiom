# Icons

We use **Google Material Symbols** icons with our Optimizely Design System via the `@optiaxiom/icons` package.

## Installation

### Install package

```sh npm2yarn
npm install @optiaxiom/icons
```

### Basic usage

Simply import the icon components and use them in your application.

```tsx
import { IconDelete } from "@optiaxiom/icons";

<IconDelete />;
```

Icons accept a `size` prop to control their dimensions:

```tsx
import { IconDelete } from "@optiaxiom/icons";

<IconDelete size="sm" />;
```

Each icon also has a filled variant via the `filled` prop:

```tsx
import { IconDelete } from "@optiaxiom/icons";

<IconDelete filled />;
```
