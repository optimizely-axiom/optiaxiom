# Flex to Stack migration

## Why Stack?

### Background

The [Flex](/components/flex) component has non-standard defaults that cause confusion for developers and AI tools. [Stack](/components/stack) is the replacement with CSS-standard defaults that align with how flexbox works in standard CSS.

### Key differences

| Property      | Flex (deprecated) | Stack (new)          |
| ------------- | ----------------- | -------------------- |
| flexDirection | "column"          | "row" (CSS standard) |
| gap           | "16"              | no default           |

## Migration examples

### Vertical stacking (most common old usage)

```tsx
// Before
<Flex>
  <div>1</div>
  <div>2</div>
</Flex>

// After
<Stack flexDirection="column" gap="16">
  <div>1</div>
  <div>2</div>
</Stack>
```

### Horizontal layout

```tsx
// Before
<Flex flexDirection="row">
  <div>1</div>
  <div>2</div>
</Flex>

// After
<Stack>
  <div>1</div>
  <div>2</div>
</Stack>
```

Note: Since Stack defaults to `flexDirection="row"`, you don't need to specify it explicitly.

### With custom gap

```tsx
// Before
<Flex gap="8">
  <div>1</div>
  <div>2</div>
</Flex>

// After
<Stack flexDirection="column" gap="8">
  <div>1</div>
  <div>2</div>
</Stack>
```

### Horizontal with custom gap

```tsx
// Before
<Flex flexDirection="row" gap="8">
  <div>1</div>
  <div>2</div>
</Flex>

// After
<Stack gap="8">
  <div>1</div>
  <div>2</div>
</Stack>
```

### Complex layouts with alignment

```tsx
// Before
<Flex alignItems="start" flexDirection="row" gap="12" justifyContent="space-between">
  <div>Left</div>
  <div>Right</div>
</Flex>

// After
<Stack alignItems="start" gap="12" justifyContent="space-between">
  <div>Left</div>
  <div>Right</div>
</Stack>
```

## Automated migration

### Using the codemod

Run the codemod (available in v1.8.0):

```bash
npx @optiaxiom/codemod flex-to-stack src/
```

The codemod will:

- Transform `<Flex>` → `<Stack flexDirection="column" gap="16">`
- Transform `<Flex flexDirection="row">` → `<Stack>`
- Transform `<Flex flexDirection="column">` → `<Stack flexDirection="column">`
- Update imports: `import { Flex }` → `import { Stack }`
- Preserve all other props and children

## Manual migration

### Step-by-step checklist

If you prefer to migrate manually, follow these steps:

1. **Import Stack instead of Flex:**

   ```tsx
   // Before
   import { Flex } from "@optiaxiom/react";

   // After
   import { Stack } from "@optiaxiom/react";
   ```

2. **Update component usage:**
   - If you had no `flexDirection` prop: add `flexDirection="column" gap="16"`
   - If you had `flexDirection="row"`: remove it (that's the Stack default)
   - If you had `flexDirection="column"`: keep it
   - If you had no `gap` prop: add `gap="16"` (or your desired gap)

3. **Test your changes:**
   - Verify layouts render correctly
   - Check responsive breakpoints still work
   - Ensure alignments are as expected

## Common patterns

### Form layouts

```tsx
// Before
<Flex gap="16">
  <Input placeholder="Email" />
  <Input placeholder="Password" />
  <Button>Submit</Button>
</Flex>

// After
<Stack flexDirection="column" gap="16">
  <Input placeholder="Email" />
  <Input placeholder="Password" />
  <Button>Submit</Button>
</Stack>
```

### Button groups

```tsx
// Before
<Flex flexDirection="row" gap="8">
  <Button>Cancel</Button>
  <Button>Save</Button>
</Flex>

// After
<Stack gap="8">
  <Button>Cancel</Button>
  <Button>Save</Button>
</Stack>
```

### Card content

```tsx
// Before
<Flex gap="12">
  <Heading>Title</Heading>
  <Text>Description</Text>
  <Button>Action</Button>
</Flex>

// After
<Stack flexDirection="column" gap="12">
  <Heading>Title</Heading>
  <Text>Description</Text>
  <Button>Action</Button>
</Stack>
```
