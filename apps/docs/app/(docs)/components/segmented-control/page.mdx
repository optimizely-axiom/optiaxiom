# SegmentedControl

Toggle buttons for switching between different values or views.

<Links component="segmented-control" />

## Documentation

### Usage

Use `SegmentedControl` and `SegmentedControlItem` components together to create sets of toggle buttons.

`SegmentedControlItem` extends the [Button](./button/) component.

<Demo component="segmented-control/usage" />

### Anatomy

```tsx
import { SegmentedControl, SegmentedControlItem } from "@optiaxiom/react";

export default () => (
  <SegmentedControl>
    <SegmentedControlItem />
  </SegmentedControl>
);
```

### Controlled

Use the `value` and `defaultValue` props to toggle between controlled and uncontrolled usage. And combine it with `onValueChange` to listen for changes to the state.

## Accessibility

### Keyboard interactions

| Key                                                                                 | Description                                                            |
| :---------------------------------------------------------------------------------- | :--------------------------------------------------------------------- |
| <Kbd>Tab</Kbd>                                                                      | Moves focus to either the pressed item or the first item in the group. |
| <Flex flexDirection="row" gap="8"><Kbd>Space</Kbd> <Kbd>Enter</Kbd></Flex>          | Activates/deactivates the item.                                        |
| <Flex flexDirection="row" gap="8"><Kbd>ArrowDown</Kbd> <Kbd>ArrowRight</Kbd></Flex> | Moves focus to the next item in the group.                             |
| <Flex flexDirection="row" gap="8"><Kbd>ArrowUp</Kbd> <Kbd>ArrowLeft</Kbd></Flex>    | Moves focus to the previous item in the group.                         |
| <Flex flexDirection="row" gap="8"><Kbd>Home</Kbd> <Kbd>End</Kbd></Flex>             | Moves focus to the first/last item.                                    |
