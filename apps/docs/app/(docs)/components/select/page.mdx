# Select

Single select combobox widget to allow selection from a fixed set of options.

<Demo component="select/usage" />

## Guide

### Structure

Select works with lists of items provided via the `items` prop. The basic structure includes the main component provider, a trigger, and a content.

- `Select`
- `SelectTrigger`
- `SelectContent`

The trigger can be provided a `placeholder` to display in case no values have been selected yet.

<Demo component="select/overview/usage" />

### Content

The content popover will be empty until we render the same items we provided to the `items` prop. We can render items using `SelectRadioItem`.

<Demo component="select/overview/content-usage" meta="{18,20}" />

## Documentation

### Anatomy

```tsx
import {
  Select,
  SelectContent,
  SelectGroup,
  SelectItem,
  SelectLabel,
  SelectRadioItem,
  SelectSeparator,
  SelectTrigger,
} from "@optiaxiom/react";

export default () => (
  <Select>
    <SelectTrigger />

    <SelectContent>
      <SelectItem />
      <SelectRadioItem />

      <SelectGroup>
        <SelectLabel />
        <SelectItem />
        <SelectRadioItem />
      </SelectGroup>

      <SelectSeparator />
    </SelectContent>
  </Select>
);
```
