# Select

Single select combobox widget to allow selection from a fixed set of options.

## Documentation

### Usage

<Demo component="select/field-usage" />

### Anatomy

```tsx
import {
  Select,
  SelectContent,
  SelectGroup,
  SelectLabel,
  SelectRadioItem,
  SelectSeparator,
  SelectTrigger,
} from "@optiaxiom/react";

export default () => (
  <Select>
    <SelectTrigger />

    <SelectContent>
      <SelectRadioItem />

      <SelectGroup>
        <SelectLabel />
        <SelectRadioItem />
      </SelectGroup>

      <SelectSeparator />
    </SelectContent>
  </Select>
);
```

### Structure

Select works with lists of items provided via the `items` prop. The basic structure includes the main component provider, a trigger, and a content.

- `Select`
- `SelectTrigger`
- `SelectContent`

Items can be an array of strings or it can be an array of objects.

The trigger can be provided a `placeholder` to display in case no values have been selected yet.

<Demo component="select/usage" />

### Content

By default content popover will be populated using the `items` prop. We can also manually set the children of `SelectContent` for more granular control.

### Controlled

Use the `value` and `defaultValue` props to toggle between controlled and uncontrolled usage. And combine it with `onValueChange` / `onChange` to listen for changes to the state.

Use the `name` prop (and optionally `ref` and `onChange`) to integrate `Select` with forms.
