# DateRangePicker

Calendar popover that lets user pick date ranges.

<Links component="date-range-picker" />

## Documentation

### Usage

<Demo component="date-range-picker/usage" />

### Anatomy

```tsx
import {
  DateRangePicker,
  DateRangePickerContent,
  DateRangePickerTrigger,
} from "@optiaxiom/react";

export default () => (
  <DateRangePicker>
    <DateRangePickerTrigger />
    <DateRangePickerContent />
  </DateRangePicker>
);
```

### Controlled

Use the `value` and `defaultValue` props to toggle between controlled and uncontrolled usage. And combine it with `onValueChange` to listen for changes to the state.

<Demo component="date-range-picker/controlled-usage" />

### Addons

Use the `addonBefore` and `addonAfter` prop on `DateRangePickerContent` to add content to either side of the calendar.

## Accessibility

### Keyboard interactions

| Key                                                                                                                                                                                      | Description                                                                                                                                                    |
| :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <Kbd>Space</Kbd>                                                                                                                                                                         | When focus is on `DateRangePickerTrigger`, opens the calendar and focuses the selected day (or today's date). When focus is on a day, selects the focused day. |
| <Kbd>Enter</Kbd>                                                                                                                                                                         | When focus is on `DateRangePickerTrigger`, opens the calendar and focuses the selected day (or today's date). When focus is on a day, selects the focused day. |
| <Flex gap="8"><Flex flexDirection="row" gap="8"><Kbd>ArrowUp</Kbd> <Kbd>ArrowDown</Kbd></Flex><Flex flexDirection="row" gap="8"><Kbd>ArrowRight</Kbd> <Kbd>ArrowLeft</Kbd></Flex></Flex> | Navigate between days in the calendar.                                                                                                                         |
| <Flex flexDirection="row" gap="8"><Kbd>PageUp</Kbd> <Kbd>PageDown</Kbd></Flex>                                                                                                           | Navigate between months in the calendar.                                                                                                                       |
| <Flex flexDirection="row" gap="8"><Kbd modifiers="shift">PageUp</Kbd> <Kbd modifiers="shift">PageDown</Kbd></Flex>                                                                       | Navigate between years in the calendar.                                                                                                                        |
| <Kbd>Esc</Kbd>                                                                                                                                                                           | Closes the calendar popover and moves focus back to `DateRangePickerTrigger`.                                                                                  |
