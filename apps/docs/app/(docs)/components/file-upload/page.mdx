# FileUpload

Capture file input from users with drag and drop.

<Links component="file-upload" />

<Alert intent="information" mt="16">
  This component is only for capturing file input from users and does not handle
  actual uploading of files or making HTTP requests.
</Alert>

## Documentation

### Usage

<Demo component="file-upload/usage" />

### Anatomy

```tsx
import {
  FileUpload,
  FileUploadDropzone,
  FileUploadList,
  FileUploadTrigger,
} from "@optiaxiom/react/unstable";

export default () => (
  <FileUpload>
    <FileUploadDropzone />
    <FileUploadTrigger />
    <FileUploadList />
  </FileUpload>
);
```

### Multiple files

`onFilesDrop` receives every selected file (`File[]`), so a single drop or selection can capture multiple files at once.

### File list

Use `FileUploadList` to preview the captured files. Pass the file items via the `items` prop, and handle removal with `onRemove`, which receives the removed item.
