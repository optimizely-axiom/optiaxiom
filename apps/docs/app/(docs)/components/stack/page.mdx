# Stack

A flexbox layout component for stacking items horizontally or vertically.

<Links component="stack" />

## Documentation

### Usage

The Stack component uses CSS-standard defaults: `flexDirection="row"` (horizontal layout) with no default gap. You must explicitly set the gap value.

<Demo component="stack/usage" />

### Direction

Use `flexDirection` prop to control whether children are stacked vertically or horizontally. By default `flexDirection` is set to `row` (CSS standard).

Use `flexDirection="row"` or `flexDirection="row-reverse"` to stack items horizontally:

<Demo component="stack/horizontal-usage" />

Use `flexDirection="column"` or `flexDirection="column-reverse"` to stack items vertically:

<Demo component="stack/vertical-usage" />

### Gap

Use `gap` prop to control the gap between stack items. Unlike the deprecated Flex component, Stack has no default gap value - you must specify it explicitly.

<Demo component="stack/gap-usage" />

### Alignment

Use `alignItems` and `justifyContent` prop to control the layout and alignment of stack items.

The Stack component provides smart `alignItems` defaults:

- When `flexDirection="row"`: defaults to `alignItems="center"`
- When `flexDirection="column"`: defaults to `alignItems="stretch"`

<Demo component="stack/alignment-usage" />

### Migrating from Flex

Stack is the replacement for the deprecated Flex component. Key differences:

| Property      | Flex (deprecated) | Stack (new)          |
| ------------- | ----------------- | -------------------- |
| flexDirection | "column"          | "row" (CSS standard) |
| gap           | "16"              | no default           |

See the [migration guide](../../guides/stack-migration/) for more details.
