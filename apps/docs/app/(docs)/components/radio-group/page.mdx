# RadioGroup

Basic control to allow selecting only one item from a set.

<Links component="radio" />

## Documentation

### Usage

Use `RadioGroup` and `Radio` components together to create sets of radio buttons.

<Demo component="radio-group/usage" />

### Label

The `children` prop on `Radio` is used as the label of the radio control.

<Demo component="radio-group/label-usage" />

### Controlled

Use the `value` and `defaultValue` props on `RadioGroup` to toggle between controlled and uncontrolled usage. And combine it with `onValueChange` to listen for changes to the value.

<Demo component="radio-group/controlled-usage" />

### Form integration

Use `Controller` from react-hook-form to integrate RadioGroup with form validation.

<Demo component="radio-group/form-integration-usage" />

### Disabled state

Enable the `disabled` prop on the `RadioGroup` to toggle the disabled state of all the radio controls.

<Demo component="radio-group/disabled-group-usage" />

You can also enable the `disabled` prop on each `Radio` to toggle the disabled state of each radio control.

<Demo component="radio-group/disabled-usage" />

### Description

Use the `description` prop to add helper text after the label.

<Demo component="radio-group/description-usage" />

### Error state

Wrap `RadioGroup` in a `Field` component and use the `error` prop to display validation errors.

<Demo component="radio-group/error-usage" />

### Common patterns

#### Conditional fields

Use the selected radio value to conditionally show additional fields.

<Demo component="radio-group/pattern-conditional-usage" />

## Accessibility

### Keyboard interactions

<KeyboardTable
  interactions={[
    {
      keys: ["Tab"],
      description: "Moves focus into and out of the radio group",
    },
    {
      keys: ["Space"],
      description: "Selects the focused radio if not already selected",
    },
    {
      keys: ["ArrowDown", "ArrowRight"],
      description: "Moves focus to and selects the next radio",
    },
    {
      keys: ["ArrowUp", "ArrowLeft"],
      description: "Moves focus to and selects the previous radio",
    },
  ]}
/>

RadioGroup automatically handles accessibility:

- Uses proper `role="radiogroup"` and individual `role="radio"` attributes
- Supports `aria-checked` for selected state
- Groups radios with proper keyboard navigation
