# Menu

Display a dropdown menu.

## Documentation

### Usage

Use the `Menu`, `MenuTrigger`, and `MenuContent` components to build out menus.

<ax-demo name="menu/usage" />

### Controlled

Use the `open` and `defaultOpen` props to toggle between controlled and uncontrolled usage. And combine it with `onOpenChange` to listen for changes to the state.

<ax-demo name="menu/controlled-usage" />

### Non modal usage

By default scrolling and interactions with outside elements is disabled when menu is open. Set `modal` prop to `false` in order to render a non-modal dropdown.

<ax-demo name="menu/modal-usage" />

## Trigger

### Customize trigger

By default we use the `Button` component for the menu trigger which accepts all of the existing button props.

<ax-demo name="menu/trigger-usage" />

### Composition

We can also completely change the trigger by using `asChild` and passing our own component.

<ax-demo name="menu/trigger-composition-usage" />

## Content

### Menu position

Use the `align` and `side` props on `MenuContent` to change the default position of the dropdown.

<ax-demo name="menu/position-usage" />

## Item

### Appearance

Use the `colorScheme` prop to control the appearance of items.

<ax-demo meta='/colorScheme="danger"/' name="menu/appearance-usage" />

### Disabled items

Use the `disabled` prop on menu items to disable them.

<ax-demo meta="/disabled/" name="menu/disabled-usage" />

### Label

Use the `MenuLabel` component to render a read-only label.

<ax-demo meta="/MenuLabel/" name="menu/usage" />

### Selection

Use the `onSelect` prop to handle selection of items.

<ax-demo name="menu/select-usage" />

### Left and right decorators

Use the `startDecorator` and `endDecorator` prop to add icons, text, or any other element to the start or end of the menu items.

<ax-demo meta="/startDecorator/ /endDecorator/" name="menu/decorator-usage" />
